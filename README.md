# Xstock
Xstock: A 15-Year Longitudinal Dataset Bridging Social Media and B2B Financials for Explainable Stock Forecasting

Xstock is a 15-year longitudinal benchmark (2011–2025) that connects Business-to-Business (B2B) social media discourse with institutional financial performance. By integrating 220,000 corporate tweets with granular SEC 10-K revenue segment filings, this project provides a framework for Explainable AI (XAI) in quantitative finance.
